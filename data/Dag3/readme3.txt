För varje nytt system som testas, gör en mapp med ett textdokument 
som beskriver kort om det är en eller två puckar, deras vikter och 
radie, vilken storhet som testas.

Mappen benämns med [SystemTal]_[Dimension][Storhet]
t.ex. S1_2Drm --> Första systemet som testas, 2 dimensioner, rörelsemängd

Text dokumentet innehåller identifierande information så som:

STORHET [rm], [e], [rmm]

Puck1 vikt radie Puck2 vikt radie
_____ ____ _____ _____ ____ _____
---------------------------------------------------------------------


Därefter kan mappen fyllas upp med filer om puck 1 och 2
(eller bara en puck som vid referensen)

Filnamnen kan vara följande:

velocity: magnitude eller components

angular velocity: magnitude eller components

System positional puck 1 and 2
----------------------------------------------------------------------

Exempel:

S1_P1_vel_comp_1 --> System 1, Puck 1, velocity components, test 1
S20_P2_ang_vel_mag_2 --> System 20, Puck 2, angular velocity magnitude, test 2
Sys15_pos_3 --> System 15 positional data, test 3
